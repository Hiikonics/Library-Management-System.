# Campus Room Booking System

> A web-based room booking system for university campus.

## Status

Phase 1 (Authentication) — ready for development.

## Quick Start

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
